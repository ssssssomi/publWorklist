Box

Box는 레이아웃과 정보 묶음을 위한 기본 컨테이너입니다. 카드와 달리 클릭/이동 목적이 아닌 경우에 사용합니다.

1. Basic box

가장 기본적인 박스입니다. 영역 구분, 설명 묶음 등에 사용합니다.

2. Box with header

박스 제목

제목 + 본문 구조의 박스입니다. 설정 영역, 정보 블록에 적합합니다.

3. Info / Notice box

안내

회원 정보는 마이페이지에서 수정할 수 있습니다.

4. Warning / Error box

주의

입력하신 정보 중 누락된 항목이 있습니다.

오류

서버 오류로 요청을 처리하지 못했습니다.

5. Emphasis box

이 영역은 중요한 정보를 강조할 때 사용합니다.
