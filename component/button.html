Button

버튼은 button 태그를 기본으로 사용합니다. 색상/테두리/배경은 var(--token) 기반이며, 프로젝트 투입 시 _variables.scss의 토큰 값만 교체하면 전체 버튼 톤이 변경됩니다.

1. Variants

* “링크 이동”은 <a> 사용, “동작 실행”은 <button> 사용을 권장합니다.

2. Sizes

* 터치 타깃은 최소 44px을 권장합니다. (사이즈는 프로젝트 기준에 맞게 토큰으로 조절)

3. Icon / Icon only

* 아이콘 전용 버튼은 반드시 aria-label로 의미를 제공합니다.

4. Disabled / Loading

* 로딩 상태에서는 aria-busy="true" 사용을 권장합니다. (실서비스에서는 클릭 방지를 위해 JS로 disabled 처리하거나 요청 중 중복 클릭을 막아주세요.)
