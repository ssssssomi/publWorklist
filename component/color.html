Color

프로젝트에 투입될 때는 색상 토큰(CSS 변수)만 교체해서 사용합니다. 컴포넌트에서는 var(--token)만 사용하고, HEX/rgba 하드코딩은 지양합니다.

  • 색상 변경은 한 곳에서: assets/scss/base/_variables.scss의 :root 토큰만 수정
  • 용도 우선: “예쁜 색”이 아니라 “역할(배경/텍스트/강조/테두리/글라스)” 기준으로 사용
  • 접근성: 텍스트/배경 조합은 명도 대비(WCAG) 충족을 전제로 선택

Core tokens

Background

  • BG
    Canvas --pub-bg-0

    전체 배경(최하위 캔버스)

Accent

  • Aa
    Accent --pub-accent

    핵심 강조(버튼, 링크, 포커스)

  • Soft
    Accent soft --pub-accent-soft

    배경 하이라이트, hover 배경, 배지

  • Weak
    Accent weak --pub-accent-weak

    아주 약한 강조(그래프/분할/광원)

  • Peach
    Peach --pub-peach

    보조 포인트(그라데이션/배경 톤)

  • Coral
    Coral --pub-coral

    보조 포인트(상태/강조 영역)

  • Amber
    Amber --pub-amber

    보조 포인트(주의/하이라이트)

Glass

  • Glass
    Glass --pub-glass

    기본 글래스 배경(카드/패널)

  • Strong
    Glass strong --pub-glass-strong

    명확한 카드 배경(가독성 우선)

  • Weak
    Glass weak --pub-glass-weak

    아주 약한 영역(서브 패널)

Stroke

  • Stroke
    Stroke --pub-stroke

    기본 테두리(카드/구분선)

  • Strong
    Stroke strong --pub-stroke-strong

    강조 테두리(활성/선택)

Text

  • Text
    Text --pub-text

    본문 기본 텍스트

  • Dim
    Text dim --pub-text-dim

    서브 텍스트(설명/보조)

  • Mute
    Text mute --pub-text-mute

    비활성/메타(주의: 대비 체크)

Focus

  • Focus
    Focus --pub-focus

    키보드 포커스 링(삭제 금지)

프로젝트에 적용하는 방법

  1. 토큰 파일만 바꾼다 — assets/scss/base/_variables.scss의 값 교체
  2. 컴포넌트는 토큰만 사용 — color: var(--pub-text) 형태로만 작성
  3. 브랜드 컬러가 달라도 구조 유지 — 토큰 이름(역할)은 그대로 두고 값만 변경
:root {
  --pub-bg-0: #ffffff;

  --pub-accent: #1e62ff;
  --pub-accent-soft: rgba(30, 98, 255, 0.18);
  --pub-accent-weak: rgba(30, 98, 255, 0.10);

  --pub-glass: rgba(255, 255, 255, 0.72);

  --pub-stroke: rgba(0, 0, 0, 0.12);

  --pub-text: rgba(0, 0, 0, 0.92);
  --pub-text-dim: rgba(0, 0, 0, 0.70);
  --pub-text-mute: rgba(0, 0, 0, 0.52);

  --pub-focus: #1e62ff;
}
