List

리스트는 시맨틱 태그(ul / ol / li)를 유지하고, 표현 방식은 data-type 속성으로 제어합니다. 색상/간격은 토큰 기반으로 관리합니다.

1. Bullet list (data-type="bul")

  • 기본 불릿 리스트 항목
  • 두 번째 항목입니다
  • 세 번째 항목입니다

2. Number list (data-type="num")

  • 첫 번째 단계
  • 두 번째 단계
  • 세 번째 단계

* 숫자 순서가 중요한 경우에만 사용합니다.

3. Dash list (data-type="dash")

  • 대시(-) 형태의 설명 리스트
  • 주의사항, 보조 설명에 적합
  • 문장이 길어져도 가독성을 유지

4. Mixed content

  • 항목 안에 강조 텍스트나 링크를 포함할 수 있습니다.
  • 보조 설명 텍스트도 자연스럽게 들어갑니다.

5. Nested list

  • 상위 항목
    • 하위 항목 1
    • 하위 항목 2
  • 다음 상위 항목

6. Disabled / muted list

  • 비활성/안내용 리스트
  • 회색 톤으로 정보만 전달
