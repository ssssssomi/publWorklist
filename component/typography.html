Typography

프로젝트 공통 타이포 가이드입니다. 제목/본문/보조문구/링크/리스트/코드 등 기본 텍스트 요소를 일관된 규칙으로 제공합니다.

1. Heading Scale

문서 구조는 시맨틱 태그(h1~h6)를 우선 사용하고, 디자인 크기 조절은 클래스(옵션)로 보조합니다.

H1 2.8rem / 900 / 1.2

H1. 페이지 대표 타이틀

H2 2.4rem / 900 / 1.25

H2. 섹션 타이틀

H3 2.0rem / 800 / 1.3

H3. 서브 섹션 타이틀

H4 1.8rem / 800 / 1.35

H4. 그룹 타이틀

H5 1.6rem / 700 / 1.4
H5. 소제목
H6 1.4rem / 700 / 1.45
H6. 캡션/보조 제목

2. Body / Helper Text

Body (Default)

기본 본문 텍스트입니다. 가독성을 위해 line-height를 확보하고, 문장 길이가 길어져도 읽기 편한 대비로 유지합니다.

Body Small

작은 본문/설명 문구에 사용합니다. 너무 연한 색상은 피하고 대비를 확보합니다.

Helper / Hint

입력 가이드/힌트는 보조 텍스트로 분류합니다. 정보 전달이 목적이면 색만 연하게 하지 말고 문구도 명확하게 작성합니다.

Caption

캡션은 표/이미지/부가 설명에 사용합니다. (예: “* 단위: 원, VAT 포함”)

3. Inline Text Elements

이 문장에는 강조(strong), 기울임(em), 하이라이트, 그리고 기본 링크가 포함되어 있습니다.

약어는 WCAG처럼 abbr에 title을 제공합니다.

5. Quote / Code Block

“접근성은 옵션이 아니라 기본값이다. 키보드/스크린리더 사용자를 포함한 모두가 동일하게 정보를 얻을 수 있어야 한다.”

— Publishing Guide

6. Utility Classes (Optional)

“시맨틱 우선” 원칙을 지키되, 디자인에서 크기만 조절해야 하는 경우에만 유틸리티를 사용합니다.

.txt-xl — 큰 텍스트

.txt-lg — 강조 본문

.txt-md — 기본 본문

.txt-sm — 보조 본문

.txt-xs — 캡션
